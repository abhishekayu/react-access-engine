# Installation

## Package Manager

Install with your preferred package manager:

```bash
# npm
npm install react-access-control

# pnpm
pnpm add react-access-control

# yarn
yarn add react-access-control

# bun
bun add react-access-control
```

## Requirements

- **React** 18+ or 19+
- **TypeScript** 5.0+ (recommended, but not required)

## DevTools (Optional)

The DevTools overlay is a separate package to keep the core bundle small:

```bash
npm install @react-access-control/devtools
```

## Bundle Size

The library is tree-shakeable with zero runtime dependencies:

| Package                          | ESM    | CJS    |
| -------------------------------- | ------ | ------ |
| `react-access-control`           | ~23 KB | ~24 KB |
| `@react-access-control/devtools` | ~30 KB | ~32 KB |

Import only the hooks and components you use — unused code is eliminated by your bundler.

## TypeScript Setup

For the best type inference, use `as const` on your config arrays:

```ts
const config = defineAccess({
  roles: ['admin', 'editor', 'viewer'] as const,
  permissions: {
    admin: ['*'],
    editor: ['posts:read', 'posts:write'],
    viewer: ['posts:read'],
  },
} as const);

// Now you get full type inference:
type Roles = InferRoles<typeof config>; // 'admin' | 'editor' | 'viewer'
type Perms = InferPermissions<typeof config>; // 'posts:read' | 'posts:write' | '*'
```
